LynString is a desktop app (macOS, Windows, Linux) that scans your Android project, finds missing or outdated translations, and writes them straight back into your `res/values-*/strings.xml` files. No copy-paste. No external tools.

> 7-day free trial · [See pricing](www.lynstring.dev) · No API key required

## Features

- Detects missing translations across all locales and fills them with one click
- Detects modified strings and offers to re-translate just those
- Preserves placeholders (`%1$s`, `%2$d`, etc.) and HTML formatting
- Escapes special characters correctly for Android
- Supports protected keys and project-specific context
- Writes directly to your project files
- Powered by Gemini AI — 100+ languages supported
- Dark / light mode 🌙☀️

## Privacy

No personal information or source code is ever collected or shared. The app sends only crash logs and basic usage statistics. Your string content is sent to the AI provider for translation, and nothing else. See the full [privacy policy](https://www.lynstring.dev/privacy).
